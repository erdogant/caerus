Installation
################


Pypi
**********************

.. code-block:: console

    # Install from Pypi:
    pip install caerus

    # Force update to latest version
    pip install -U caerus


Github source
************************************

.. code-block:: console

    # Install directly from github
    pip install git+https://github.com/erdogant/caerus


Uninstalling
################


Remove installation
**********************

Note that the removal of the environment will also remove the ``caerus`` installation.

.. code-block:: console

    # Install from Pypi:
    pip uninstall caerus
